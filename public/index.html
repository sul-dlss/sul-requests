Admin Features

Request Definitions

This part of the application defines various kinds of requests by library, current location, and other criteria. The required information is passed to the request application from Socrates or SearchWorks. Each request definition can have its own text for request forms and its own selection of fields that will appear on the form.

Request Tests

Pickup Libraries

Requests can be picked up at different sets of libraries, depending on various criteria. This part of the application provides keys based on data in Symphony records, each of which can be associated with a set of pickup libraries to display on a request form. The keys are based on various kinds of information found in Symphony records, such a substrings from certain Symphony location strings.
